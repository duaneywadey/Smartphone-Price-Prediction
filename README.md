# Instructions
How to run this project? 

* Step 1. 
```
git clone https://github.com/duaneywadey/Smartphone-Price-Prediction.git
```

* Step 2. 
```
venv\Scripts\Activate
```

* Step 3. 
```
pip install -r requirements.txt
```

* Step 4.
```
set FLASK_ENV=development
```

* Step 5.
```
flask run  
```
